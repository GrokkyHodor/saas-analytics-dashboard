Pulse — SaaS Analytics Dashboard
A fully interactive SaaS analytics dashboard built with vanilla HTML, CSS, and JavaScript. Designed to give SaaS founders and product teams a real-time view of their most important business metrics.

Features

Real-time activity feed — live event stream that updates automatically
Interactive revenue chart — switch between MRR, ARR, and New MRR views
Plan distribution donut chart — visual breakdown of customers by plan tier
Key metric cards — MRR, active customers, churn rate, and ARPU at a glance
CSV export — one-click data export for reporting
Responsive layout — adapts cleanly to different screen sizes
Zero dependencies — no frameworks, no build step, just open and run

Tech Stack

HTML5 & CSS3 (custom properties, CSS Grid, Flexbox)
Vanilla JavaScript (ES6+)
Chart.js for data visualisation
Google Fonts — Bricolage Grotesque & JetBrains Mono

Getting Started
No installation required. Just clone the repo and open the file:
bashgit clone https://github.com/GrokkyHodor/saas-analytics-dashboard.git
cd saas-analytics-dashboard
open index.html
Or simply download index.html and open it in any browser.
Use Cases
This dashboard is built for SaaS businesses that need to track:

Monthly and Annual Recurring Revenue (MRR / ARR)
Customer growth and churn
Plan distribution across pricing tiers
Real-time subscription events

Customisation
All data is defined in plain JavaScript arrays at the top of the <script> block — easy to swap in real data from any API or backend.
